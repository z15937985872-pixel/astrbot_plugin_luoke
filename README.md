## 安装说明
   
   1.从插件市场中安装此插件
   
   2.在插件目录中安装 playwright和aiohttp 依赖
   
     pip install playwright aiohttp
	 
   3.安装Chrome浏览器，在地址栏输入chrome://version/，查看你的Google Chrome版本
   
   4.安装chromedriver，国内镜像：https://registry.npmmirror.com/binary.html?path=chrome-for-testing/，在其中找到你的Google Chrome版本号，或是离得最近的版本号，选择适合你的版本，选择chromedriver-win64.zip（或是其他版本），下载后解压，将chromedriver.exe文件放在astrbot\AstrBot-master中
   
   5.重启astrbot
   
## 6.指令说明
     /查精灵 精灵名称

	 /配队详情 123（填括号里的ID）
     /查技能 技能名称
     /抽精灵 （每日一次）
	 /查蛋组 精灵名称
	 /查孵蛋 公/母+精灵名称 希望孵出的精灵名称
